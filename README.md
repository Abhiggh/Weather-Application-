# Weather Application 🌤️
A real-time weather web application built with JavaScript, HTML, and CSS that fetches live weather data from a public API and displays it through a clean, responsive UI.

# 📌 About the Project
This project demonstrates how to consume a public REST API, handle JSON responses, and display dynamic data on a responsive web interface. Users can search for any city and instantly see current weather conditions

# 🛠️ Technologies Used
- Languages: JavaScript, HTML, CSS
- API: OpenWeatherMap API (REST)
- Concepts: API Integration, JSON Parsing, Responsive Design, DOM Manipulation
